<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added bulk domain selection: Select multiple domains during search and add them to cart at once.
- Introduced price history charts: View past pricing trends for any domain before purchase.

## Improvements

- Optimized search performance: Queries now return results `<2s` even for popular TLDs like `.com` and `.io`.
- Enhanced mobile responsiveness: Dashboard adapts seamlessly to tablets and phones.

## Bug Fixes

- Fixed cart persistence issue where items disappeared on page refresh.
- Resolved payment gateway timeout errors during high-traffic periods.

</Update>

<Update label="2024-09-20" description="v1.1.0" tags={["feature", "bugfix"]}>

## New Features

- Integrated WHOIS privacy toggle: Enable privacy protection at checkout with one click.
- Added domain availability alerts: Set notifications for your watchlisted domains.

## Improvements

- Upgraded search filters: Filter by price range, length, and keyword relevance more precisely.

## Bug Fixes

- Corrected expiration date display for premium domains.
- Patched vulnerability in auction bidding endpoint.

</Update>

<Update label="2024-08-10" description="v1.0.0" tags={["feature", "breaking"]}>

## New Features

- Core domain search engine: Explore millions of available domains across 500+ TLDs.
- Secure checkout flow: Purchase domains instantly with multiple payment methods.
- User dashboard: Manage purchases, watchlists, and transfers.

## Breaking Changes

- Updated API authentication: Use `{API_KEY}` header instead of query params.

## Bug Fixes

- Initial stability improvements for search indexing.

</Update>
